1. Mengapa menggunakan CSS dan JavaScript dalam pemrograman web?

CSS dan JavaScript digunakan dalam pemrograman web karena keduanya berperan penting dalam pembuatan tampilan dan interaktivitas situs web. CSS (Cascading Style Sheets) bertanggung jawab untuk mengatur tata letak, desain, dan estetika halaman web, seperti warna, font, margin, dan pengaturan layout lainnya. JavaScript, di sisi lain, digunakan untuk memberikan interaktivitas, seperti validasi form, animasi, dan manipulasi elemen HTML. Kombinasi keduanya memberikan pengalaman pengguna yang lebih baik dan tampilan yang menarik.

3. Jelaskan apa saja keuntungan menggunakan JavaScipt dan CSS?

Keuntungan menggunakan JavaScript dan CSS di antaranya adalah: • Efisiensi dan pengalaman pengguna: JavaScript membuat halaman web responsif terhadap interaksi pengguna, dan CSS meningkatkan estetika serta tata letak, yang membuat tampilan menjadi menarik dan mudah diakses. • Cache pada browser: File CSS dan JavaScript eksternal dapat disimpan dalam cache browser, yang mempercepat waktu muat halaman ketika pengguna kembali mengunjungi situs. • Keteraturan kode: CSS dan JavaScript eksternal membantu dalam menjaga struktur kode yang terorganisir dengan baik, memudahkan pembaruan dan pemeliharaan di masa mendatang.

4. Jealskan apa saja keuntungan menggunakan Bootstrap?

Keuntungan menggunakan Bootstrap antara lain: • Responsif: Bootstrap memungkinkan tampilan web menyesuaikan dengan perangkat (mobile, tablet, desktop) secara otomatis. • Komponen siap pakai: Bootstrap menyediakan banyak komponen seperti navbar, card, tombol, dan lain-lain yang mudah digunakan. • Kustomisasi yang mudah: Bootstrap mudah dikustomisasi dan memiliki banyak tema yang bisa langsung diterapkan. • Kompatibilitas antar-browser: Bootstrap telah diuji agar kompatibel dengan berbagai browser populer.
